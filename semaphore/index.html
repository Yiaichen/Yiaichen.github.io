# 4. 总结

Semaphore semaphore = new Semaphore (1) 其实只是初始化多少个许可
acquire () 相当于动态的减少许可，相应的 release () 可以动态的增加许可
Semaphore 提供的限制并发线程的功能，此功能在默认的 synchronized 种是不提供的
